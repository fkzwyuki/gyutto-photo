■ ぎゅっと写真（photo-compressor）の使い方

【これは何？】
スマホのカメラで撮った写真やスクリーンショットを「ぎゅっと」軽くして、
そのままスマホに保存・保管できるアプリです。
index.html 1ファイルだけで動きます。サーバー不要・通信なし（すべて端末内で処理）。

【スマホでの使い方】
1. index.html をスマホのブラウザ（Safari / Chrome）で開く
   - 例1: このファイルを自分宛にAirDrop/LINE等で送って開く
   - 例2: GitHub Pages などに置いて URL で開く（おすすめ）
2. 「共有」→「ホーム画面に追加」すると、アプリのように起動できます

【操作の流れ】
1. 「写真・スクショを選ぶ」→ カメラロールから複数選択OK
   （「カメラで撮る」でその場で撮影も可能）
2. 自動で圧縮され、1枚ごとに「元サイズ → 圧縮後サイズ（−○%）」を表示
3. 保存先を選ぶ
   - ⬇ 保存   … 端末にダウンロード（iPhoneは共有シートから「画像を保存」で写真アプリへ）
   - 📦 保管庫へ … アプリ内の保管庫（ブラウザのIndexedDB）に保管
   - 📤 共有   … LINEやメール等へそのまま送る（対応ブラウザのみ）

【設定】
- サイズ（長辺）: そのまま / 2048 / 1600 / 1280 / 1024px
- 画質: 40〜95%（おすすめ80%）
- 形式: JPEG（互換性◎）/ WebP（より小さい）
※ 圧縮後のほうが大きくなる写真は、自動で元データのまま扱います
※ EXIFの回転情報は反映されます（位置情報などのEXIFは削除されます）

【保管庫について】
- タブの「📦 保管庫」で一覧表示。タップで拡大 → 保存/共有/削除
- データは端末のブラウザ内に保存されます。
  ブラウザのサイトデータを消すと消えるため、大切な写真は「保存」で
  写真アプリにも書き出しておいてください。

【動作環境】
iOS Safari 15+ / Android Chrome / PCの各種ブラウザ
