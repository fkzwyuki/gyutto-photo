# ぎゅっと写真 📷

スマホのカメラで撮った写真やスクリーンショットを「ぎゅっと」軽くして、そのままスマホに保存・保管できるWebアプリです。

**▶ アプリを開く: https://fkzwyuki.github.io/gyutto-photo/**

スマホのブラウザで開いて「共有 → ホーム画面に追加」すると、アプリのように使えます。

## 特徴

- **端末内で完結** — サーバーへのアップロードなし。圧縮はすべてブラウザ内で処理
- **複数枚まとめて圧縮** — カメラロールから複数選択、またはその場でカメラ撮影
- **設定を選べる** — 長辺サイズ（そのまま〜1024px）・画質（40〜95%）・形式（JPEG / WebP）
- **3つの保存先** — 端末に保存（写真アプリへ）／アプリ内の保管庫（IndexedDB）／共有シートでLINE等へ
- **賢い動作** — EXIFの回転を反映。圧縮後のほうが大きくなる場合は元データを自動保持

## 開発

`index.html` 1ファイルのみで動作します（依存なし・ビルド不要）。
`main` ブランチへのpushでGitHub Pagesに自動デプロイされます。

開発の本体は [SClaude](https://github.com/fkzwyuki/SClaude) リポジトリの `photo-compressor/` で行い、このリポジトリは公開用です。
